# EBM Math Calculator Guide

An interactive Evidence-Based Medicine calculator for medical students and healthcare professionals learning clinical research statistics.

## Features

- **Confidence Interval Interpreter** - Determine statistical significance
- **Risk Calculator** - RR, ARR, NNT for cohort studies and RCTs
- **Odds Calculator** - Odds Ratio for case-control studies
- **Diagnostic Testing** - Sensitivity, Specificity, PPV, NPV, Likelihood Ratios
- **Clinical Significance** - Effect size vs. statistical significance concepts
- **PDF Export** - Save calculations as study notes

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## For Students

1. Identify your study type (RCT, cohort, case-control, diagnostic)
2. Select the appropriate calculator tab
3. Enter data from your research paper
4. Click Calculate to see results with interpretations
5. Use the Q&A sections to understand concepts
6. Export to PDF for your notes
